Mix

  • . - Any character except newline
  • [] - Character class, match any character listed in the class.
  • () - Grouping and capturing.
  • \1 - Internal variable representing the content of the first capture.
  • | - Alternation

Anchors

  • ^ - At the beginnin of the regex, require match at the beginning of the string.
  • ^ - As the first character in a character class, negate the characer class.
  • $ - At the end of the regex, require match at the end of the string.
  • \b - Beginning or end of word.

Quantifiers

  • * - Quanitier: 0 or more times.
  • + - Quanitier: 1 or more time.
  • ? - Quanitier: 0 or 1 times.
  • {n, m} - Quanitier: Between n and m times.
  • {n,} - Quanitier: n or more times.
  • {n} - Quanitier: exactly n times.

Character classes

  • \d - Digit. The same as [0-9].
  • \w - Word character. The same as [a-zA-Z0-9_].
  • \s - White space.
  • \D - Non digit. The same as [^\d].
  • \W - Non word character. The same as [^\w].
  • \S - Non white space. The same as [^\s].
