Python - match

The re package also provides a match method. It is similar to the search method but it only matches at the beginning of the string. As if there was an implicit ^ at the beginning of the regular expression. 

import re
m = re.match(r'a', 'bac')
print m # null

m = re.match(r'b', 'bac')
print m.group(0) # b
