In Python the re package provides the Regular Expression support. The search method of the package will get a regular expression and a string. It will return a match object or None, if there was no match. The match object has a group method. Passing 0 to it will return the substring that was matched. It is usually better to write the Regular Expression as a "raw" string (preceding it with an r). 

import re
match = re.search(r'a', 'bac')
if match:
   print match.group(0)   # a

m = re.search(r'd', 'bac')
print m    # None
