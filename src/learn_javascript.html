JavaScript

In JavaScript the strings and the variables containing strings have methods related to pattern matching. The patterns themselves are enclosed in slashes, just as it is done in Perl. The two main methods used are "search", that returns the location of the first match, and "match", that will return an array with more information.

If there is no match, the "search" method will return -1, while the "match" method will return "null".

You would probably write something like this: 

  var res = str.match(/REGEX/);
  if (res) {
     // do something with res
  }
