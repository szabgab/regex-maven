Regex Maven

Welcome to Regex Maven

The Regex Maven is a mobile application that will help you learn to use Regular Expressions (or regexes). It will make it easy for you to parse text in any programming languages (for example Perl, Python, PHP, Ruby, JavaScript, Java, just to name a few.). It will help you practice with a built-in regex evaluator, and it will let you test yourself with a quiz.

It is currently in an early beta version, but you can already use it.
